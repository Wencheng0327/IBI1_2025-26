I think the difficulty of this practical assignment is about the same as before, but it's a bit harder. Because it involves both probability models and drawing graphs at the same time. The probability model requires you to pay attention to how each step changes, so it's a little bit tricky to understand the changes within each time period.
The first task was the easiest part, because the logic was clear: susceptible people become infected, and infected people recover. 
The vaccination model of second task was also interesting, because it showed clearly how increasing vaccination reduced the size of the infected population.
The spatial SIR model of third task was the hardest part. It was more challenging, because I had to think about neighbours, boundaries of the grid, and how to update the population without changing values too early. However, it was also the most interesting part, because the heat map made the spread of infection easier to visualise.
The most important thing I learned was how to turn a real biological problem into a probabilistic model and draw a plot of them in Python. I also learned that running the same stochastic code several times can give slightly different results, which is an important feature of this kind of model.
